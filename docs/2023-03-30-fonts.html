Fonts

I always want the best fonts for my terminal, my text editor, my ... But I'm not satisfied easily, so I keep trying new fonts. Prefer free fonts of course :D

TLDR:

On macOS, prefer San Francisco. Remember each time macOS release new version/new update, you should download again to get (maybe) latest version.

Otherwise:

  • Use Inter for display.

Prefer:

  • If macOS choose OTF, otherwise choose TTF format.
  • If macOS ignore hinting, otherwise use hinting.
  • Choose static font over variable font.

My standard for choosing fonts:

  • Support Vietnamese
  • Support bold, italic
  • Has dotted zero, because I dislike slashed zero

Manually install fonts:

  • macOS directory: ~/Library/Fonts
  • Linux directory: ~/.local/share/fonts
