Can not go without vim, with extension
[Vim](https://marketplace.visualstudio.com/items?itemName=vscodevim.vim)

```json
{
  "vim.enableNeovim": true,
  "vim.neovimPath": "/change/your/path/here",
  "vim.textwidth": 120,
  "vim.whichwrap": "<,>,[,]"
}
```
